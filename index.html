Picard

A set of tools (in Java) for working with next generation sequencing data in the BAM (http://samtools.sourceforge.net) format.

Introduction

Picard is a set of Java command line tools for manipulating high-throughput sequencing data (HTS) data and formats.

Sequence Alignment/Map (SAM) format is a generic format for storing large nucleotide sequence alignments. SAM and related file formats are described in the hts-specs page.

Picard comprises Java-based command-line utilities that manipulate SAM files, and a Java API (HTSJDK) for creating new programs that read and write SAM files. Both SAM text format and SAM binary (BAM) format are supported.

Picard is implemented using the HTSJDK Java library, supporting accessing of common file formats, such as SAM and VCF, used for high-throughput sequencing data.

To get help using and developing Picard, please use the GATK Forums.

Enjoy!

Quick Start

Step 1: Download Software

The Picard command-line tools are packaged as a single executable jar file.

  • If you would just like to use the toolset to analyze your data and do not require the source code, you may download the jar file by clicking this button: Latest Release

  • For developers, the source code and building instructions are available on: GitHub

Download the pre-compiled executable jar file packaged as "picard-tools-2.x.x.zip" . Open the downloaded package and place the folder containing the jar file in a convenient directory on your hard drive (or server).

For the tools to run properly, you must have Java 1.8 installed. It is recommended that you download the latest version of the Java Development Kit (JDK) and not the Java Runtime Environment (JRE). The JDK package can be found here: JDK

To check your java version by open your terminal application and run the following command: 

java -version
If the output indicates version "1.8", you are good to go. If not, you may need to reinstall your JDK.

Step 2: Test Installation

Test your executable file by navigating in your terminal application to the folder containing the jar file. Once there, type the following command: 
java -jar picard.jar -h
You should see a complete list of all the tools in the Picard program.

Before Asking For Help

Before posting to the Forum, please do the following:

  • Try the latest version of Picard
  • See if your problem is discussed in the Frequently Asked Questions
  • Search the GATK Forums to see if your problem was previously discussed.
  • Run Picard ValidateSamFile MODE=SUMMARY. Attempt to resolve or at least understand any problems reported.

Getting Help

Picard is supported through the GATK Forums. Join the GATK Forums to get support for Picard programs among other tools supported by the Broad Institute.

When asking a question about a problem, please include the following:

  • Command line(s) you used.
  • Program console output and metrics files. Repetitive console output may be abbreviated.
  • Entire stack trace if one was produced.
  • Version of JVM are you using (obtained by running 'java -version').
